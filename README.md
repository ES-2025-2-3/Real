# Real

Stirling-PDF é uma ferramenta completa para manipular arquivos PDF, totalmente gratuita, open-source e que pode ser executada localmente via Docker.  Foi projetada para não armazenar arquivos os PDFs ficam no lado do cliente sempre que possível, quando precisam ir ao servidor, ficam somente na memória durante o processamento. Prezando a privacidade dos dados do usuário. 
O Stirling-PDF reúne mais de cinquenta ferramentas para lidar com PDFs. Ele organiza documentos, permitindo juntar, dividir, extrair, remover, reorganizar, rotacionar e ajustar páginas. Também cria layouts especiais, gera páginas contínuas e faz cortes precisos.
Converte praticamente qualquer formato para PDF, desde imagens, HTML e Markdown até eBooks e documentos comuns. E vice-versa transformando PDFs em Word, imagens, HTML, tabelas, arquivos vetoriais, PDF/A e outros formatos. 
Oferece edição completa: OCR, limpeza de scans, inserção e extração de imagens, edição visual, metadados, comparação, numeração, flatten, remoção de anotações e páginas em branco.
Permite assinar digitalmente, proteger com senha, remover senhas, ocultar informações, mudar permissões, validar assinaturas, adicionar marcas d'água e limpar elementos sensíveis. 
Comprime arquivos, cria pipelines automáticos, ajusta cores, renomeia documentos, separa fotos, repara PDFs, mostra scripts internos, divide por capítulos, edita sumários e aplica efeitos de digitalização.
